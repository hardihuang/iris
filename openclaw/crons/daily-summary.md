# 每日工作总结定时任务

## 任务说明
- **执行时间**: 每天晚上 8 点 (20:00)
- **时区**: Asia/Shanghai
- **功能**: 
  1. 总结当天工作内容
  2. 创建/更新 MD 文档
  3. 发送飞书消息通知

## Cron 表达式
```
0 20 * * *
```

## 执行流程
1. 获取当天日历事件
2. 获取当天任务完成情况
3. 获取当天重要消息/对话
4. 生成总结文档 (Markdown 格式)
5. 保存到 memory/ 目录
6. 发送飞书消息给用户
